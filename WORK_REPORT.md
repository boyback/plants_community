   - 霸王鞭属：霸王鞭、红雀珊瑚

   - 豹皮花属：豹皮花、大花犀角、紫龙角

   - 白粉藤属：翡翠阁、酒瓶兰

6. **风信子科** (Hyacinthaceae)
   - 风信子属：风信子

   - 芦荟属：库拉索芦荟、不夜城芦荟、木立芦荟、珍珠芦荟
   - 十二卷属：玉露、寿、万象、玉扇、条纹十二卷

8. **鸭跖草科** (Commelinaceae)
   - 紫竹梅属：紫竹梅、吊竹梅

## 访问路径

### 前端页面
- 摄影大赛列表：`/contests`
- 大赛详情：`/contests/[id]`
- 晒图广场：`/shaitu`
- 相册详情：`/album/[id]`

### API 端点
- `GET /api/contests` - 获取大赛列表
- `POST /api/contests` - 创建大赛
- `GET /api/contests/[id]` - 获取大赛详情
- `PATCH /api/contests/[id]` - 更新大赛
- `DELETE /api/contests/[id]` - 删除大赛
- `GET /api/contests/[id]/entries` - 获取参赛作品
- `POST /api/contests/[id]/entries` - 提交作品
- `POST /api/contests/entries/[entryId]/vote` - 投票
- `DELETE /api/contests/entries/[entryId]/vote` - 取消投票
